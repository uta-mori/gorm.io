Вклад в GORM

С вашей помощью GORM станет лучше! Вот несколько вещей, которые вы можете сделать:

Делиться информацией о GORM с другими Golang разработчиками.

  • Распространяйте информацию через социальные каналы, или просто расскажите о GORM людям, которые могли бы быть заинтересованы
  • Помогите начинающим разобраться в GORM, отвечая на вопросы в Stack Overflow, Github Issues and Chat Rooms
  • Пишите статьи, записывайте видео, и т.д…. (Не забывайте добавить ссылку на комьюнити)
  • Помогайте GORM репозиторию стать популярнее на GitHub Starring и Watching

Писать документацию

  • Сделайте документацию GORM лучше, путем написания нового содержания или исправления существующего материала

Переводить этот сайт

Вы можете перевести текущий сайт здесь https://translate.gorm.io

Чтобы перевести GORM на ваш язык, вам нужно будет отправить запрос в Github issues, после того как перевод будет добавлен в https://translate.gorm.io.

Пишите код

  • Создайте pull request on Github на устранение багов, опубликованных на Github Issues
  • Пишите open source плагины для GORM. (Не забывайте добавлять ссылку на комьюнити)
